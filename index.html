Projects

MOSSE Filters (Correlation Filter Tracking)

• A high-performance object tracking system built in C++ (with Python bindings) using MOSSE correlation filters and OpenCV.
• Utilizes 2 filters per tracked object (long-term + short-term) and adaptively updates them based on detection confidence (PSR).
• Initialized with a user-defined ROI; continuously tracks across frames and handles occlusions and appearance changes.
• Designed for webcam or Raspberry Pi camera setups.
• Future enhancements in the repo: dynamic track addition, expanding search when object exits frame, and mode-specific tracking (e.g. human, vehicle).

Bird Classification

• Developed a convolutional neural network (CNN) using transfer learning with E]icientNet-B2 to
identify and classify 200 bird species from photographic images.
• Fine-tuned pretrained weights on a custom dataset with data augmentation (random rotations, flips,
and color perturbations) to enhance robustness under variable lighting and pose conditions.
• Implemented a PyTorch-based training pipeline with stratified splits, early stopping, and learning rate

Visual Odometry

• Implemented a monocular visual odometry system to estimate camera trajectory from image frames.
• Uses feature detection (ORB / FAST), essential matrix estimation, pose recovery, and frame-to-frame motion composition.
• Incorporated drift correction heuristics and loop detection to reduce error over time.
